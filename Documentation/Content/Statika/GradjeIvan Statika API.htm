API
GradjeIvan Statika je aplikacija napravljena na bazi funkcija sadrzanih unutar GradjeIvan API. Ovaj API daje gotove funkcije za definisanje grafickog okruzenja aplikacije, dijaloga, grafickog prikaza i unosa modela, tabelarnog prikaza podataka, proracuna sa matricama.. Razumevanje koda GradjeIvan API uslov za razumevanje kompletnog koda aplikacije GradjeIvan Statika.
Za objasnjenje GradjeIvan API-ja postoji posebno poglavlje.

Definiciju slozenijih funkcija i objekata karakteristicnih za aplikaciju GradjeIvan Statika obradjuje ovo poglavlje.
